 Como Funciona

            
    
                
  1. Carregue um arquivo Lua: Clique no botão de upload ou arraste e solte um arquivo .lua.
    2. 
                
  2. Edite as configurações: Altere login, senha, data de expiração e número de dispositivos permitidos.
    3. 
                
  3. Aplique as alterações: Clique em "Aplicar Alterações" para atualizar o código Lua.
    4. 
                
  4. Visualize o código: Use o botão "Visualizar Código" para ver as mudanças no script.
    5. 
                
  5. Faça download: Escolha entre baixar o arquivo original, modificado ou um relatório JSON.
    6. 
                
  6. Formato esperado no arquivo Lua:
                    
    -- Configurações de acesso
LOGIN = "usuario123"
SENHA = "senhaSecreta@2023"
DATA_EXPIRACAO = "31/12/2023"
DISPOSITIVOS = 3
    
                
    7.
